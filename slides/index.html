Listener

A listener is a named network location (e.g., port, unix domain socket, etc.) that can be connected to by downstream clients. Envoy exposes one or more listeners that downstream hosts connect to.

@liamandrewwhite

Route(r)

At a high level the router takes an incoming request, matches it to an upstream cluster, acquires a connection pool to a host in the upstream cluster, and forwards the request.

@liamandrewwhite

Cluster

A cluster is a group of logically similar upstream hosts that Envoy connects to. Envoy discovers the members of a cluster via service discovery.
